Functions for Array methods

Challenge en parejas. ​ Crea un modulo con funciones que repliquen, sin utilizar ninguno de los métodos ya existentes en el prototipo, los siguientes métodos de array:

    length,
    push,
    pop,
    unshift,
    shift,
    some,
    every,
    find,
    filter,
    map,
    findIndex,
    includes,
    indexOf,
    reduce,
    join
